# ⛽ Surge 油价查询面板

Surge 网络工具的油价查询面板模块，自动获取全国各地最新油价，面板中用 Unicode 方块展示油价趋势图。

## 功能特性

- 📊 **实时油价**：6种油品（0号柴油、89/92/95号汽油、-10/-20号柴油）
- 📈 **Unicode 趋势图**：▁▂▃▄▅▆▇█ 可视化展示油价走势，底部对齐
- ⏰ **调价信息**：显示最近调价日期和下次调价倒计时
- 🏙️ **多城市支持**：覆盖全国主要城市
- ⚙️ **参数化配置**：城市、油品、定时推送 cron 均可自定义
- 🔔 **定时通知**：支持每日定时推送油价信息

## 安装

### 方式一：通过 Surge 模块安装（推荐）

在 Surge 中添加模块，URL 填写：

```
https://raw.githubusercontent.com/GitHubWBB/Surge/main/oil_price/oil_price.sgmodule
```

### 方式二：本地安装

1. 下载 `oil_price_panel.js` 和 `oil_price.sgmodule` 到同一目录
2. Surge → 模块 → 安装新模块 → 选择 `.sgmodule` 文件

## 配置参数

编辑模块参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `city` | `cd` | 城市拼音缩写 |
| `fuel` | `95` | 关注的油品编号 |
| `cronexp` | `0 8 * * *` | 定时推送 cron 表达式 |
