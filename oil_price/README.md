# ⛽ 油价查询面板

Surge 网络工具的油价查询面板模块，自动获取全国各地最新油价，面板中用 ASCII 方式展示油价趋势图。

## 功能

- 📊 **实时油价**：6种油品（0号柴油、89/92/95号汽油、-10/-20号柴油）
- 📈 **ASCII 趋势图**：直观展示油价走势
- ⏰ **调价提醒**：显示下次调价日期和倒计时
- 🏙️ **多城市支持**：覆盖全国主要城市
- ⚙️ **参数化配置**：城市、油品可自定义

## 安装

在 Surge 中添加模块，URL 填写：

```
https://raw.githubusercontent.com/GitHubWBB/Surge/main/oil_price/oil_price.sgmodule
```

## 配置参数

编辑模块中的 `argument` 参数：

```
argument=city=cd&fuel=92
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `city` | `cd` | 城市缩写 |
| `fuel` | `92` | 关注的油品编号 |
